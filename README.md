# 🎲 Decisive Gamer (电子游戏选择困难症拯救者)

> "So many games, so little time. Let the code decide."

**Decisive Gamer** 是一个专为“电子阳痿”和“选择困难症”玩家打造的 Web 应用。面对 Steam 库里成百上千款吃灰的游戏不知道玩哪个？只需轻轻一点，让算法为你决定今晚的冒险。

## ✨ 核心功能 (Features)

- **🎲 随机命运之轮**：一键从游戏库中随机抽取一款游戏，专治“打开列表发呆半小时”。
- **📚 游戏库展示**：精美的卡片式布局，展示已拥有的游戏收藏、游玩时长及个人锐评。
- **🚀 Steam 直连**：集成 Steam AppID，随机选中后可直接跳转商店或启动游戏。
- **⚡ 极速响应**：基于 Vite + React 构建，丝滑流畅。

## 🛠️ 技术栈 (Tech Stack)

- **构建工具**: [Vite](https://vitejs.dev/) - 极速的前端构建工具
- **核心框架**: **React** (通过 `main.jsx` 入口判断)
- **语言**: JavaScript / JSX
- **样式**: CSS / Tailwind CSS (根据实际情况调整)
